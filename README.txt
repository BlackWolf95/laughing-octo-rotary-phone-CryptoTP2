To execute the given code, extract the tar ball and please do the following on the terminal:
1. make : to compile the C source codes and create the executables.
2. ./targ : to execute the test cases alongwith the implementation of the Speck48/96 encryption and finding a fixed point(s) when it is used for the Davies Meyer hash construction (narrow piped).

The implementation of the collision detection is yet to be finished (Exercise 2: The attack).
